# 🦝 Trash Panda Loops - Web Design Specification

## 🎨 Design Philosophy

**Theme:** Modern Dark Gaming Aesthetic with Playful Raccoon Elements
**Vibe:** Cozy midnight heist, neon-lit kitchen, retro-futuristic
**Target Audience:** Casual gamers, roguelike fans, ages 16-35
**Mood:** Fun, mysterious, slightly mischievous, addictive

---

## 🌈 Color Palette

### Primary Colors
```
Deep Night Blue:     #1a1d2e (Main background)
Midnight Purple:     #2d1b3d (Secondary background)
Neon Cyan:          #00d9ff (Primary accent, buttons, highlights)
Electric Purple:     #b537f2 (Secondary accent, special elements)
Warm Orange:        #ff6b35 (Call-to-action, warnings)
```

### Secondary Colors
```
Soft Cream:         #f4f1de (Text, light elements)
Muted Gray:         #9ba4b5 (Secondary text)
Dark Charcoal:      #0f1419 (Cards, panels)
Success Green:      #4ecca3 (Positive feedback)
Warning Yellow:     #ffd93d (Alerts)
Danger Red:         #ff6b6b (Errors, detection)
```

### Game-Specific Colors
```
Raccoon Gray:       #8b8b8b (Raccoon character)
Snack Gold:         #ffd700 (Common snacks)
Rare Blue:          #6c5ce7 (Rare items)
Legendary Pink:     #fd79a8 (Legendary items)
Kitchen Beige:      #d4a574 (Kitchen elements)
```
